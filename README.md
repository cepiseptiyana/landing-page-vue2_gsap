# TRICKS Scroll Toggle

A visually engaging interactive website with smooth scroll-triggered animations built with **Vue.js** and **GSAP**. Inspired by the [TRICKS Scroll Toggle demo](https://tr-scroll-toggle.webflow.io/), this project showcases dynamic text and image animations that respond to scrolling.

## Features

- Smooth scroll-triggered animations using **GSAP ScrollTrigger**.
- Large typography with **centered scroll alignment**.
- Image scaling animations synchronized with scrolling.
- Scrollable text sections that update visual elements in real-time.
- Fully responsive layout for desktop and mobile screens.
- Reusable Vue components for text and image sections.

---

## Technologies Used

- **Vue.js 2** – reactive frontend framework for component-based development.
- **GSAP** – animation library for smooth, performant scroll-based animations.
  - **ScrollTrigger plugin** – controls scroll-based triggers and pinning.
  - Optional **ScrollToPlugin** for smooth scrolling within sections.
- **HTML5 & CSS3** – semantic markup and styling.
- SCSS for structured styling and variables.
